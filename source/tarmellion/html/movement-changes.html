Movement Changes

There have been some small, but far-reaching modifications done to movement.

  1. Ship Speed

    Each ship type has its own speed. The exact values for each ship can be seen here.

  2. Sailable land

    The terrain called lake is primarily land, but it can be sailed by ships as if it was ocean.

  3. Modified Summon Wind spell

    The Summon Wind spell now adds a bonus to sailing and flying speed equal to its skill level independant of the ship type used.

  4. Tunnels

    There is an underground terrain type called tunnels that has comes in three different types, that the player cannot tell apart. They look the same, but one type needs 1 movement point, the other two movement points and the last three movement points. This terrain feature was added to compensate for the lack of bad weather underground. Together with the walls (unavailable movement directions) this should balance underground and surface.

  5. Unsailable land

    The terrain called volcano cannot be entered by ship.

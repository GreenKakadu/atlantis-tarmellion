Ressources of Tarmellion

Here are listed all available ressources of the game Tarmellion.

The three types of buildings listed are for the three of the building types established in Tarmellion.
Building 1 gives a fixed bonus of a specified amount of the given ressource in the hex where the building is created. This only works in a hex where there is already some of the given ressource. The amount dwindles with each new building of the same type.
Building 2 improves the availability of the given ressource by 25 %. Each subsequent building gives only half of that (12%, 6%, ...). This also only works in hexes with the specified ressource.
Building 3 is necessary for the production of the listed ressource. Producers must be inside this building if they want to produce the ressource at all. Each such building can host a fixed amount of producers.

Tool specifies the item that helps produce this ressource. A unit with the needed skill will get a bonus of man months listed behind the tool if he uses that tool while producing the ressource.

Value is the amount of silver one of these items is worth. This is used when calculating the price of the ressource in markets or the amount of the ressource gotten in monster spoils. It also is used when the positive effect of trade (buying AND selling) for the city growth is calculated each month.
